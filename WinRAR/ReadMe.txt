   Ce package contient l'archiveur multifonctions WinRAR.

   Les caractéristiques de WinRAR sont les suivantes :

 * WinRAR introduit un algorithme de compression spécifique. Il permet 
   un meilleur taux de compression que d'autres utilitaires de compression 
   sur PC, surtout pour les fichiers exécutables, les bibliothèques 
   d'objets, les fichiers textes de grande taille, etc.

 * WinRAR offre en option, un algorithme de compression fortement 
   optimisé pour les données multimédia.

 * WinRAR supporte les fichiers et les archives d'une taille pouvant 
   aller jusqu'à 9 223 372 036 854 775 807 octets, à peu près 9000 Po. 
   Le nombre de fichiers pouvant être archivés est quasiment illimité.

 * WinRAR fournit un support complet pour les archives RAR et ZIP 2.0, 
   il désarchive également les archives CAB, ARJ, LZH, TAR, GZ, ACE, 
   UUE, BZ2, JAR, ISO, Z, 7Z.

 * WinRAR supporte la sécurité fichier et les flux de données NTFS.

 * WinRAR comporte deux interfaces, une classique sous Windows et une 
   pour le mode ligne de commande.

 * WinRAR donne la possibilité de créer des archives "Solide" dont le 
   taux de compression est augmenté de 10% à 50% par rapport aux méthodes 
   traditionnelles, particulièrement dans le cas d'un grand nombre de 
   petits fichiers.

 * WinRAR offre la possibilité de créer et modifier les archives SFX en 
   utilisant le module SFX par défaut et des modules externes.

 * WinRAR offre la possibilité de créer des archives multi-volumes SFX.

 * WinRAR offre un grand nombre de fonctions: définition d'un mot de passe, 
   ajout de commentaires dans les archives ou les fichiers. Il est même
   possible de réparer les archives endommagées physiquement.
